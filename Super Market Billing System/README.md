# Super Market Billing System

A simple console-based Super Market Billing System developed in C++.

This project helps manage grocery items, generate customer bills, and update item quantities after every purchase. It is built using basic C++ concepts such as classes, file handling, and object-oriented programming.

## Features

* Add new items with price and quantity
* View all available items
* Generate customer bills
* Automatically update stock after purchase
* Store item data using text files

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* File Handling
* Visual Studio

## How to Run

1. Clone or download this repository.
2. Open the project in Visual Studio or any C++ IDE.
3. Build and run the program.
4. Follow the menu displayed in the console.

## Project Structure

* `SuperMarketBillingSystem.cpp` – Main source code
* `Bill.txt` – Stores item details
* `Bill Temp.txt` – Temporary file used while updating stock
