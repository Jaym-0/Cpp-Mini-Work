# Tic Tac Toe

A simple console-based Tic Tac Toe game developed in C++.

This project allows two players to play Tic Tac Toe in the terminal. Players can choose their markers, take turns, and the program automatically checks for a winner or a draw.

## Features

* Two-player gameplay
* Choose between X and O
* Input validation for moves
* Detects winner and draw
* Play multiple rounds without restarting the program

## Technologies Used

* C++
* Functions
* Arrays
* Basic Game Logic

## How to Run

1. Clone or download this repository.
2. Open the project in Visual Studio or any C++ IDE.
3. Build and run the program.
4. Follow the instructions displayed in the console.
